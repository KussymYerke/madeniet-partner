# Как добавлять новые бренд-страницы

Проект теперь поддерживает страницы партнёров через один универсальный компонент:

```jsx
<PartnerPage page={partnerPages.hyundai} />
```

## Где менять данные

Все данные партнёров лежат в:

```txt
src/data/partnerPages.js
```

Hyundai — первый пример. Чтобы добавить новый бренд:

1. Скопируйте объект `hyundai`.
2. Переименуйте ключ, например в `sony` или `magnum`.
3. Измените:
   - `slug`
   - `brand`
   - `color`
   - `logo`
   - `images`
   - тексты внутри `content.ru` и `content.kz`
   - `moodboard.items`
   - пакеты в `formats.packages`

После этого страница автоматически откроется по URL:

```txt
/sony
/magnum
/bi-group
```

если такой ключ есть в `partnerPages`.

## Главный принцип

Не копируем HTML на 10–20 страниц. Меняем только данные бренда, а визуальная структура остаётся общей через:

```txt
src/components/PartnerPage.jsx
```

## Как использовать Pinterest-референсы

Pinterest нужен только как визуальный ориентир: атмосфера, цвет, типографика, композиция, ритм и ощущение журнала. Это не готовые шаблоны и не макеты Madeniet. Мы не копируем чужой дизайн — финальный визуал собирается оригинально под бренд.

Сейчас mood board работает даже без готовых изображений: карточки рисуются через CSS, чтобы страница не выглядела пустой.

Когда будут скриншоты/экспорты Pinterest-референсов:

1. Создайте папку:

```txt
public/assets/moodboard
```

2. Положите туда изображения, например:

```txt
public/assets/moodboard/hyundai-cover.jpg
public/assets/moodboard/hyundai-spread.jpg
public/assets/moodboard/hyundai-digital.jpg
```

3. В `src/data/partnerPages.js` добавьте поле `image` в нужную карточку:

```js
{
  tag: "cover system",
  title: "Премиальная обложка",
  text: "Много воздуха, крупная типографика...",
  variant: "cover",
  image: "/assets/moodboard/hyundai-cover.jpg",
  alt: "Референс премиальной обложки"
}
```

Если `image` не указан, останется аккуратная CSS-заглушка.
